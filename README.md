# VPS Security Testing

Repository ini berisi dokumentasi dan hasil pengujian keamanan pada **Virtual Private Server (VPS)** dengan fokus pada pengujian serangan jaringan, meliputi **Port Scanning**, **Denial of Service (DoS)**, dan **Brute Force Attack**.  
Pengujian dilakukan sebagai bagian dari pembelajaran dan evaluasi keamanan sistem jaringan.

Pengujian yang dilakukan dalam project ini meliputi:

### 1. Port Scanning
Pengujian untuk mengetahui port-port yang terbuka pada VPS serta layanan yang berjalan di dalamnya.  
Hasil pengujian digunakan untuk mengidentifikasi potensi celah keamanan.

### 2. Denial of Service (DoS)
Pengujian untuk mengevaluasi ketahanan VPS terhadap serangan yang membanjiri server dengan trafik berlebihan sehingga layanan menjadi tidak tersedia.

### 3. Brute Force Attack
Pengujian untuk mengukur keamanan sistem autentikasi dengan mensimulasikan percobaan login berulang menggunakan berbagai kombinasi username dan password.
